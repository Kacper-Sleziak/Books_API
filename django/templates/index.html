BOOKS API DOCS

PK in endpoints stands for primary key



/api_spec


Return information about API

/books


Return books for given params (title, authors[], from, to, acquired), return all books for zero parameters

/books/create


Create a book for given in-body parameters

/books/{pk}


Return book with the given id

/books/{pk}


Edit book with the given id

/books/{pk}


Delete book with the given id

/import


Import all books for given in-body author from google API
